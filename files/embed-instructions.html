1 Add the widget to your website

Paste this single line anywhere before the closing </body> tag on every page you want the chat to appear. That's it — no other files, no npm, no configuration needed.

<!-- Big Hat Lawn Chat Widget — paste before </body> --> <script src="https://cdn.jsdelivr.net/gh/kalelra/512ai@main/files/widget-bighatlawn.js" defer></script>
Note for web developers: The CDN URL is served via jsDelivr (global CDN backed by Cloudflare + Fastly). Loads in <50ms. The defer attribute ensures it never blocks page rendering.

2 Platform-specific instructions

🔷

WordPress

Go to Appearance → Theme Editor → footer.php and paste the script tag just before </body>. Or use the WPCode plugin and add it as a "Footer" snippet.

⬛

Squarespace

Go to Settings → Advanced → Code Injection → Footer and paste the script tag there. It applies site-wide automatically.

🔶

Wix

Open the Wix Editor → Add → Embed Code → Embed HTML. Paste the script tag. Then drag the element to the bottom of the page.

🟣

Webflow

Go to Project Settings → Custom Code → Footer Code and paste the script tag. Publish to apply site-wide.

🟠

Shopify

Go to Online Store → Themes → Edit Code → layout/theme.liquid and paste the script tag before </body>.

📄

Static HTML

Paste the script tag before </body> in any HTML file. Works on any standard website without any framework or build tool.

3 What visitors experience

💬

Proactive greeting

After 8 seconds on the page, a friendly bubble pops up: "Need a lawn quote? I can get you one in 2 minutes!"

🔴

Unread badge

A red badge appears on the chat button so visitors notice the greeting even without the bubble.

⏱️

Typing indicator

Animated dots show while the AI is thinking — feels like a real person responding.

📅

Booking confirmation

When a booking is completed, a green confirmation card appears and the booking is recorded in the system.

📱

Mobile-optimized

On phones, the widget becomes a full-width bottom sheet — matches native app feel.

🔁

Session memory

Conversation context is preserved if the visitor navigates between pages on the same site.

6 Optional: control widget from your site's code

After the widget loads, two global functions are available if you want to trigger it programmatically:

// Open the chat widget from any button or link on your site BHL_OPEN(); // Close the chat widget BHL_CLOSE(); // Example: open widget when "Get a Quote" button is clicked document.getElementById('quote-btn') .addEventListener('click', BHL_OPEN);
